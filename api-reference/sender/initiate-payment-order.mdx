---
title: 'Initiate Payment Order'
openapi: 'POST /sender/orders'
---

Initiate a new payment order as a sender.

---

## Headers

| Name     | Type   | Required | Description         |
|----------|--------|----------|---------------------|
| API-Key  | string | Yes      | Your API key        |

---

## Request Body

```json
{
  "amount": 1000,
  "token": "USDT",
  "rate": 1.0,
  "network": "base",
  "recipient": {
    "institution": "ABCPC",
    "accountIdentifier": "1234567890",
    "accountName": "John Doe",
    "memo": "Payment for invoice #123",
    "providerId": "provider-001",
    "metadata": {},
    "currency": "NGN"
  },
  "reference": "INV-123",
  "returnAddress": "0xabc..."
}
```

---

## Response (201)

```json
{
  "id": "order-001",
  "amount": "1000",
  "token": "USDT",
  "network": "base",
  "receiveAddress": "0xabc...",
  "validUntil": "2024-01-01T00:00:00Z",
  "senderFee": 10,
  "transactionFee": 5,
  "reference": "INV-123"
}
```

---

## Status Codes

| Code | Description                |
|------|----------------------------|
| 201  | Payment order initiated    |
| 400  | Bad request                |
| 401  | Unauthorized (invalid key) |
| 500  | Server error               |
